# 20日，10日均线支撑买点需求
- 判断股价每次最接近10或20日均线的某一天时，是不是成交量要大于前一日成交量，而且股价要涨幅较大（创业板大于6个点，主板大于4个点）
- 只统计最近3个月的出现这种情况的次数
- 输出的结果，累计时间段内接近（上下差距-1到1个点）10日或则20日均线的次数。满足接近均线出现放量大涨的次数
- 输出个格式:
```json
{
  "code":"300118",
  "near_10":[
    {
    "day":"2026-01-23",
    "rate":"当日涨幅",
    "next_rate":"第二日涨幅"
    }
  ]，
    "near_20":[
      {
    "day":"2026-01-23",
    "rate":"当日涨幅",
    "next_rate":"第二日涨幅"
    }
  ]

  
}
```
